Product Design

From the creation of the event to entrance statistics, shop configuration and billing – existing functions were optimized and new ones developed. All sections are always available, no matter whether with the smartphone or on desktop.

Sales Client

With the Progressive Web App, event organizers can sell tickets in the ticket office and directly at the event – optimized for tablets and smartphones.

Individual shop

I have conceived and designed a responsive online shop that can be easily customized. The order can be placed on a single screen directly in the shopping cart. Customers don't have to remember passwords anymore – if a ticket is lost, an e-mail with a magic link is sufficient.

UX and UI Design

Together with the product team, new functions are planned. These are verified with prototypes in user tests and then prepared for the developers and finalized in design reviews.
